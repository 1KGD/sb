# Implementation

STARBLOOM is written in [rust](https://rust-lang.org). The graphics (in this version) are handled by [macroquad](https://macroquad.rs).

> [!NOTE]
> **Fun Fact:** This is technecally the *second* version of STARBLOOM. The first version was a very incomplete python project using [pygame](https://pygame.org). This version is already much more complete than then the original ever was.
